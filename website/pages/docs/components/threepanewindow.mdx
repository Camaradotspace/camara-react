## Props

ThreePaneWindow properties

| Property | Description                                                              | Type      | Default | Version |
| -------- | ------------------------------------------------------------------------ | --------- | ------- | ------- |
| docs     | Used to render the browser window on the docs differently from Storybook | `boolean` | `false` |         |
