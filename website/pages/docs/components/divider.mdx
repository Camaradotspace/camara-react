# Divider

Divider description.

<br></br>

## Props

Divider properties

| Property | Description                       | Type                       | Default  | Version |
| -------- | --------------------------------- | -------------------------- | -------- | ------- |
| type     | What style should the line be of? | `dashed` `dotted` `normal` | `normal` |         |

<br></br>

## Usage

### Base

```jsx live noInline
import { Divider, BrowserWindow } from 'camara';

render(
  <BrowserWindow>
    <Divider />
  </BrowserWindow>
);
```

### Types

```jsx live noInline
import { Box, Divider, BrowserWindow } from 'camara';

render(
  <BrowserWindow>
    <Box>
      <Divider type="dashed" />
      <Divider type="dotted" />
      <Divider type="normal" />
    </Box>
  </BrowserWindow>
);
```
