# NavigationPane

NavigationPane description.

<br></br>

## Props

NavigationPane properties

| Property | Description                                                              | Type      | Default | Version |
| -------- | ------------------------------------------------------------------------ | --------- | ------- | ------- |
| docs     | Used to render the browser window on the docs differently from Storybook | `boolean` | `false` |         |

<br></br>

## Usage

### Base

```jsx live noInline
import { BrowserWindow, NavigationPane } from 'camara';

render(
  <BrowserWindow docs>
    <NavigationPane />
  </BrowserWindow>
);
```
