# TextArea

TextArea description.

<br></br>

## Props

TextArea properties

| Property    | Description                              | Type          | Default | Version |
| ----------- | ---------------------------------------- | ------------- | ------- | ------- |
| disabled    | Is input disabled?                       | `boolean`     | false   |         |
| helperText  | Input helper text                        | `string`      | -       |         |
| label       | Input label text                         | `string`      | -       |         |
| maxlength   | Textarea maximum length                  | `number`      | -       |         |
| name        | Input name                               | `string`      | -       |         |
| placeholder | Input placeholder text                   | `string`      | -       |         |
| required    | Is input required?                       | `boolean`     | false   |         |
| resizable   | Should textarea be resizable?            | `boolean`     | false   |         |
| rows        | How many rows should the text area have? | `number`      | 4       |         |
| wrap        | How should the text area wrap?           | `hard` `soft` | `hard`  |         |

<br></br>

## Usage

### Basic TextArea

```jsx live noInline
import { TextArea, BrowserWindow } from 'camara';

render(
  <BrowserWindow docs>
    <TextArea placeholder="Type something here" />
  </BrowserWindow>
);
```

### Helper Text

```jsx live noInline
import { Flex, TextArea, BrowserWindow } from 'camara';

render(
  <BrowserWindow docs>
    <Flex>
      <TextArea label="Description" helperText="example@gmail.com" />
      <TextArea
        label="What's on your mind?"
        helperText="must contain special characters"
      />
    </Flex>
  </BrowserWindow>
);
```

### Required

```jsx live noInline
import { Flex, TextArea, BrowserWindow } from 'camara';

render(
  <BrowserWindow docs>
    <Flex>
      <TextArea required label="First Name" />
      <TextArea required label="Last Name" />
    </Flex>
  </BrowserWindow>
);
```
