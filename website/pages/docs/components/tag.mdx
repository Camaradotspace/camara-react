# Tag

Tag description.

<br></br>

## Props

Tag properties

| Property | Description  | Type                     | Default   | Version |
| -------- | ------------ | ------------------------ | --------- | ------- |
| color    | Color of tag | `string`                 | `#BB65FF` |         |
| size     | Size of tag  | `small` `medium` `large` | `small`   |         |

<br></br>

## Usage

### Basic Tag

```jsx live noInline
import { Tag } from 'camara';

render(
  <>
    <Tag>Tag</Tag>
  </>
);
```

### Example

```jsx live noInline
import { Flex, Tag } from 'camara';

render(
  <>
    <Flex>
      <Tag color='#e61b1b'>red</Tag>
      <Tag color='#0099FF'>blue</Tag>
      <Tag color='#9b069b'>purple</Tag>
      <Tag color='#ff6347'>tomato</Tag>
      <Tag color='#068d6b'>teal</Tag>
    </Flex>
  </>
);
```
