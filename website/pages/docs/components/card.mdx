# Card

Card description.

<br></br>

## Props

Card properties

| Property | Description                                            | Type      | Default | Version |
| -------- | ------------------------------------------------------ | --------- | ------- | ------- |
| bordered | Should card have border around it or be entirely flat? | `boolean` | false   |         |
| elevate  | Should the card elevate on the z-index?                | `boolean` | false   |         |
| height   | Height of the card                                     | `string`  | -       |         |
| width    | Width of the card                                      | `string`  | -       |         |

<br></br>

## Usage

### Elevate

```jsx live noInline
import { Button, Card, Image, Text, BrowserWindow } from 'camara';

render(
  <BrowserWindow docs>
    <Card elevate>
      <Image
        src="https://source.unsplash.com/V_WBdOLOgJU/400x300"
        alt="woman in orange knit cap and blue jacket"
      />
      <Text>John Doe</Text>
      <Button variant="secondary" block pill>
        Follow
      </Button>
    </Card>
  </BrowserWindow>
);
```
